# Shamrock Mechanical Website

This repository contains the static website for **Shamrock Mechanical**, the public-facing brand of Shamrock Mechanical Solutions Inc.

The site is built with plain HTML and CSS so it can be hosted easily on GitHub Pages or another static hosting service.

## Pages

- `index.html` - Home page
- `services.html` - Services overview
- `projects.html` - Project types and capabilities
- `about.html` - Company overview and founders section
- `contact.html` - Contact details and mailto request form
- `styles.css` - Sitewide styling

## Brand direction

The site uses a commercial / industrial colour scheme based on:

- Deep green
- Dark green
- Charcoal
- Steel grey
- Warm off-white
- Muted gold accent

The design is intended to be clean, practical, and appropriate for a commercial and industrial mechanical contractor.
